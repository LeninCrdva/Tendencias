Se encuentran 3 carpetas en este espacio.
El primero es el servicio SOAP que se conecta a la base de datos de MySQL.
El segundo es el cliente que consume los servicios que se realizó y ofrece el servidor.
El tercero es la carpeta de dependencias, debido a que el servidor se conecta a un base de datos necesita de una librería JDBC para realizar la conexión adecuada.

Y el último archivo excluyendo al .txt es el scritp de la base de datos.
